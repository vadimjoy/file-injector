## Обзор

**AI CSS Kit** — лёгкая AI-first CSS-библиотека компонентов без зависимостей JavaScript. Разработана с акцентом на минимализм и доступность. Предоставляет полный набор компонентов форм с поддержкой светлых и тёмных тем.

### Ключевые возможности

- **Без зависимостей** — чистый CSS, никакие JavaScript-библиотеки для стилей не требуются
- **3 варианта размера** — Small (sm), Medium (md) и Large (lg) для всех компонентов
- **Полная кастомизация** — каждый визуальный аспект управляется через CSS-переменные (`--ui-*` / `--ai-*`)
- **Пресеты тем + Маппер** — `dist/themes/*.css` + CLI для конвертации JSON → CSS для кастомных тем
- **Доступность в приоритете** — совместимость с ARIA, навигация с клавиатуры, состояния фокуса
- **Современные стандарты** — CSS Grid, Flexbox, кастомные свойства, семантический HTML

## Установка

```bash
npm install ai-css-kit font-awesome
```

```html
<link rel="stylesheet" href="https://unpkg.com/ai-css-kit/dist/ai-css-kit.min.css">
```

## Быстрый старт

```html
<div class="ui-field ui-field--md">
  <label class="ui-label">Email-адрес</label>
  <input type="email" class="ui-input" placeholder="name@example.com">
  <span class="ui-helper">Адрес остаётся приватным</span>
</div>
```
