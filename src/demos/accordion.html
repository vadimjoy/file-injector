A lightweight CSS component library designed for modern AI-powered applications. It provides a set of pre-built, customisable components that work seamlessly with any framework or vanilla HTML.

Use --ai-* CSS custom properties to override tokens at any level — globally on :root, per component, or inline. No build step required.

No JavaScript is required for styling. All visual states are handled with CSS. You can add minimal JS to toggle the open class for interactive behaviour.

Flush variant

The flush variant removes outer borders and background, blending seamlessly into any container or card.

Great for settings panels and sidebars where a minimal look is preferred.

Works well alongside other flush-style components like flush lists and borderless tables.
