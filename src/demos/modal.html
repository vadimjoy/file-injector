Sizes

Small modal

This is a small modal. Use it for confirmations or quick prompts that need minimal content.

Medium modal

This is a medium modal, suitable for forms, detail views, or anything requiring a bit more space to breathe.

Large modal

This is a large modal. Use it for rich content — editors, data tables, media viewers, or multi-step flows that need more horizontal real estate.

Header variants

Title only

A modal header with just a title — clean and minimal, suitable for short focused dialogs.

Invite team member

Send an invitation link to a new team member's email address.

Use the subtitle to provide helpful context below the title without cluttering the body area.

Footer alignment

Footer end (default)

Buttons are aligned to the right — the default footer layout for most confirmation dialogs.

Footer between

Use --footer--between to push a secondary action (e.g. "Delete") to the far left.

Footer start

Use --footer--start when the primary action should sit on the left side of the footer.
