CSS Customization

Override these CSS variables to customize the appearance of all components: 

/* Primary Color */
--ui-color-primary: #4f46e5;
--ui-color-primary-hover: #4338ca;

/* Border Radius */
--ui-radius-sm: 6px;
--ui-radius-md: 8px;
--ui-radius-lg: 12px;

/* Shadows */
--ui-shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.07);

/* Transition Speed */
--ui-transition: all 0.2s ease;
